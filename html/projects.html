• Queto - AI agent focused on corporate crises

    Queto will be an intelligent autonomous system agents designed to detect, analyze, respond to and learn from crisis situations. Based on events of different natures (such as system failures, cyberattacks, or anomalous sound events), the agent executes response, planning, learning, and reporting protocols based on LLMs (LLaMA 3.2).

  • Project Integrator II (Gerenciador Patrimonial)

    The application works on mobile devices (Android and IOS), has a loading screen, a home screen with the user's data and their assets on the date they enter.   • Tasks using the FreeRTOS Kernel

    FreeRTOS is an open source kernel simulator, used for real-time applications. This project simulates the creation of 4 tasks, which are equivalent to threads. The function that executes task 3 is used to suspend the execution of task 3. The vTaskSuspend() function is used to make this interruption, so that the tasks run in a loop again, the function vTaskResume(), this is what occurs inside the for loop of task 2, in this function there is an i counter, The counter starts at 0, and increments each time Task 2 is executed by the kernel. When the counter i reaches 2, task 3 is re-harvested once again. When the counter i receives 6 as a variable value, task 4 will be executed, which terminates the kernel.
